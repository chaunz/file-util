# 说明

### 简介:

```markdown
	该项目封装了一个common-io的工具类.简化一些文件的操作,提供了一文件对象,文件夹对象,操作文件的类,操作文件夹的类.提供一些简化的操作,该项目并没有完善,日后会慢慢更新.
```



### 项目环境

- 项目是一个普通的`maven`项目

- 项目使用的`JDK`为`1.8`版本的

- 项目的依赖:

  | 依赖         | 版本    |
  | ------------ | ------- |
  | `junit`      | 4.11    |
  | `lombok`     | 1.18.10 |
  | `commons-io` | 2.4     |

- 项目结构:

  - 采用普通的`maven`项目构建,是一个`java`项目
